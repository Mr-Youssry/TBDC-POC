Visual language

  • Very light canvas with a faint pink-tinted upper band
  • Deep charcoal sidebar with cleaner active states
  • One vivid pink accent, supported by green, amber, and blue
  • Large headings, fewer words, stronger icon-led scanning
  • 8px card logic in-product, calmer shadows, disciplined surfaces

Behavioral rules

  • Tables become secondary patterns, not default first view
  • Filters are grouped and purposeful, not permanently overwhelming
  • Right rails answer the selection question only
  • Progressive disclosure replaces all-at-once reference clutter
  • Motion is subtle: rise, hover, focus, no ornamental excess
